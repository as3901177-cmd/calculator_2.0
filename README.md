# 📐 CAD Analyzer Pro v24.0

Professional DXF file analyzer for CNC and laser cutting machines.

## ✨ Features

- ✅ Accurate cut length calculation (LINE, ARC, CIRCLE, POLYLINE, LWPOLYLINE, SPLINE, ELLIPSE)
- ✅ **Smart piercing count with connectivity analysis**
- ✅ DXF visualization with error indicators
- ✅ Chain visualization (connected objects)
- ✅ Statistics by object types and colors
- ✅ CSV export
- ✅ **Parquet tessellation v9.0 for nesting optimization**
